# XcodeProjectArrangementTool
搜索项目不用的类


###### 1.第一步运行程序。
###### 2.在“UseFiles”文本框下面的，第一个文本框，输入要过滤的文件，用逗号分割。
###### 3.点击“Browse...”按钮，选择工程目录中的 .xcodeproj 文件。或者拖拽 “.xcodeproj”文件到文本框中。
###### 4.点击"search"搜索按钮. 将 UnusedFiles文框中的类去xcode中全局搜索一下，再次确定是否使用。
